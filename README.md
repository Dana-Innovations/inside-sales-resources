# ISS Repository Manager

AI-powered update tool for the Sonance Inside Sales Resources document.

## What it does

Upload the current `Inside Sales Resources.html`, describe your changes (add, remove, rename, or update links), and Claude applies them server-side and returns the updated file for download.

## Columns supported

James · Sonance | iPort · Trufig · Blaze · Trainings · Resources · SLA's & SOP's · INT'L · Coverage · Marketing

## How to use

1. Open the app
2. Upload `Inside Sales Resources.html` from your Teams Channel SharePoint
3. Describe your changes (add/remove/rename items, update URLs)
4. Click **Apply Changes with Claude**
5. Download the updated file and re-upload to SharePoint
